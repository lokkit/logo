# logo
Contains the lokkit logo

# Build images in different sizes
The file `build.sh` converts the specified svg files to several formats and sizes. The converted files
are written to `./dist/`.
```
./build.sh

ls ./dist -1
lokkit_icon_100.pdf
lokkit_icon_100.png
lokkit_icon_400.pdf
lokkit_icon_400.png
lokkit_icon_800.pdf
lokkit_icon_800.png
lokkit_logo_100.pdf
lokkit_logo_100.png
lokkit_logo_400.pdf
lokkit_logo_400.png
lokkit_logo_800.pdf
lokkit_logo_800.png
```
